Vaccine Kit Mobile Application. Bangkit Academy Capstone Project.

Vaccine Kit is 50 Best Bangkit Capstone Projects. Work to filling the requirment for graduate from bangkit. Bangkit is an approved Kampus Merdeka - Study Independent program supported by the Ministry of Education, Culture, Research, and Technology - Republic of Indonesia. Google Led program in Collaboration with Gojek, Tokopedia, and Traveloka.

Vaccine Kit Application an android app that combined machine learning, cloud and mobile development into one app.

It has feature that can help certain institutions or professionals recognize every citizen who has had vaccines by scanning the barcode on the bracelet. Also this application is able to verify it by recognizing the user's face.

We Build two apps for institutions and users in one project using kotlin-native with clean architecture and implementation android library. For the backend using retrofit, koin module and coroutine flow.

Machine learning working for face recognition feature on app build with TensorFlow Keras, using transfer learning from VGG-16.

And Connected the data through the cloud by create a REST API (including face recognition code) using the Django Rest Framework and then deploy it to an instance in GCP. Create monitoring (install cloud monitoring agent and cloud logging agent in ssh) for the instance. Create a database in Firebase and also integrating it in GCP. Create alerts for CPU Usage, Memory Usage, and Bandwidth.
